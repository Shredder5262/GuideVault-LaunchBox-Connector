This package contains GuideVault LaunchBox Connector source only.

Version 0.4.22 is the first release-hardened LaunchBox connector package. It includes scoped manual, strategy guide, and magazine sync, match review windows, status/version visibility, reduced background polling, and WebView2 reader helper deployment.

Build on a Windows machine that has LaunchBox installed:

.\scripts\Build-Plugin.ps1 -LaunchBoxRoot "C:\Path\To\LaunchBox"

Recommended local build/install:

.\scripts\Build-Plugin.ps1 -LaunchBoxRoot "C:\Path\To\LaunchBox"
.\scripts\Install-Plugin.ps1 -LaunchBoxRoot "C:\Path\To\LaunchBox" -ForceCloseLaunchBox

Create the end-user drag-and-drop release zip:

.\scripts\Package-PluginRelease.ps1 -LaunchBoxRoot "C:\Path\To\LaunchBox"

GuideVault server 0.9.260 or newer is recommended.
